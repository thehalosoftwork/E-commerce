# SOLÈNE — Luxury Footwear E-Commerce UI

A modern **luxury footwear e-commerce front-end** built with **HTML, CSS, and JavaScript**.
This project showcases a premium shopping experience with smooth animations, interactive product filtering, and a dynamic cart system.

---

## Preview

SOLÈNE is designed as a **high-end fashion brand website** focusing on elegant UI, minimal design, and smooth interactions.

Main sections include:

* Hero landing page
* Category showcase
* Featured product grid
* Interactive shopping cart
* Brand editorial section
* Customer reviews
* Newsletter subscription
* Modern footer navigation

## How It Works

1. Product data is stored inside a **JavaScript array**.
2. Products are dynamically rendered into the grid.
3. Users can filter products by category.
4. Clicking **Add to Bag** updates the cart state.
5. The cart drawer shows:

   * Product name
   * Quantity
   * Total price
6. Toast notifications provide feedback for user actions.
